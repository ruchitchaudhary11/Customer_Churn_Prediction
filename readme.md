## рҹ“Ң Project Overview

Customer churn is one of the biggest challenges faced by telecom companies. Retaining existing customers is often more cost-effective than acquiring new ones.

This project uses an **XGBoost Machine Learning model** to predict customer churn and provides an interactive dashboard built with **Streamlit**.

Users can enter customer information and instantly receive:

* Churn Prediction (Yes/No)
* Churn Probability Score
* AI-powered explanation using Gemini API
* Interactive dashboard interface

## рҹ“Ҳ Machine Learning Model

The project uses **XGBoost Classifier**, a powerful gradient boosting algorithm known for:

* High predictive accuracy
* Fast training speed
* Handling missing values effectively
* Feature importance analysis

### Model Workflow

1. Data Cleaning
2. Feature Engineering
3. Train-Test Split
